# word-guessing-game 🌱

A beginner-friendly Python word guessing game that uses loops, conditionals, lists, and user input to create an interactive guessing experience.

## Features
- Random word selection from a plant-themed word bank
- Letter-by-letter guessing
- Limited number of attempts
- Displays progress after each guess
- Win and lose conditions

## Skills Demonstrated
- Python fundamentals
- Variables and data types
- Lists
- Loops (`for` and `while`)
- Conditional statements (`if` and `else`)
- User input handling
- Debugging and problem-solving

## How to Run
1. Download or clone the repository.
2. Open the project in Python.
3. Run the program.
4. Guess letters until you reveal the word or run out of attempts.
